この[Peertube](https://joinpeertube.org/)プラグインは、お使いのPeerTubeインスタンスの動画に対してチャットシステムを提供します。

{{% notice tip %}}
左側のメニューにある言語セレクターを使って、このドキュメントをさまざまな言語で表示することができます。
{{% /notice %}}
